## üìú Usage Instructions

Once you have installed envx, you can start using it right away. Here are some basic commands:

1. **Set an Environment Variable:**
   ```
   envx set VARIABLE_NAME value
   ```
   
   Replace `VARIABLE_NAME` with your desired variable name and `value` with the value you want to assign.

2. **Get an Environment Variable:**
   ```
   envx get VARIABLE_NAME
   ```

3. **List All Environment Variables:**
   ```
   envx list
   ```

4. **Remove an Environment Variable:**
   ```
   envx remove VARIABLE_NAME
   ```

You can find more commands and examples in the documentation on the Releases page.
